# Neoincube
_Final Project of Samsung Innovation Campus 7 Stage 3 of AI and IOT intergation_ 
## Overview
Neoincube is a smart environment monitoring system designed to classify environmental temperature conditions as Too Warm, Too Cool, or Normal.
The classification is performed using a Machine Learning model integrated with IoT hardware, enabling real-time environmental analysis.
